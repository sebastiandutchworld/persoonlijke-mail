OVERVIEW

This feature make all source(origin) field clickable. Easy for users to click and navigate the origin document.
No Configuration needed. Just install module and all source document on Delivery Order, Incoming Picking, Purchase Order, Manufacturing Order are clickable.

HOW IT WORKS

=> Source Document clickable on Delivery order and open respective document (here open sales order)

=> Source Document clickable on Purchase Order and open respective document (here open sales order)

=> Source Document clickable on Incoming Receipt and open respective document (here open purchase order)

=> Source Document clickable on Manufacturing Order and open respective document (here open sales order)

=> Source Document clickable on Pick Component picking and open respective document (here open Manufacturing Order)
